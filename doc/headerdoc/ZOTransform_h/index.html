ZOTransform

Class for ratio transformation

Discussion

This class is used for transforming points from camera using ratio transformation. Description of this transform could be found in documentation of project. This class conforms to protocol TransformProtocol.


Methods

-getRightRootOfPolynomWithA:B:andC:

Returns right root of quadratic polynom

getRightRootOfPolynomWithA:B:andC:

Returns right root of quadratic polynom 

-(double)getRightRootOfPolynomWithA:(double)a B:(double)b 
        andC:(double)c;
Discussion

This method computes roots of quadratic polynom and returns the root in range 0,5. If the discriminant is negative, returns -100. If the root is out of range, returns 0.
