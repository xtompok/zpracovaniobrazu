ZOCalibrateController

Controller for calibration

Discussion

This controller implements calibration of the camera. It tells calibration view to show points, handles the timer and posts notification, when the calibration is completed.


Methods

-calibrate

Starts the calibraiton

-calibrationArray

Returns array of calibration points.

-calibrationData

Returns ZOCalibrationData object with calibration data.

-description

Returns a string with calibration points.

-setSize:

Sets the resolution of the camera.

calibrate

Starts the calibraiton 

-(void)calibrate;
Discussion

When this method is called, it takes the calibration view to fullscreen and starts calibration timer.

calibrationArray

Returns array of calibration points. 

-(NSArray *)calibrationArray;
Discussion

Returns array of four normalised ZOPoints in order: Upper left point, upper right point, lower right point, lower left point.

calibrationData

Returns ZOCalibrationData object with calibration data. 

-(ZOCalibrationData *)calibrationData;
Discussion

This methods returns the object of ZOCalibrationData, which is described in its own class.

description

Returns a string with calibration points. 

-(NSString *)description;
Discussion

This method returns a string with description of each calibration point nad its coordinates.

setSize:

Sets the resolution of the camera. 

-(void)setSize:(NSSize)aSize;
