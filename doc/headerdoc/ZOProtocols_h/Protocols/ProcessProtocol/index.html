getThePointFromImage:

Found lighted point from given image 

-(NSPoint)getThePointFromImage:(NSImage *)anImage;
Discussion

This method is never called before initWithSize. It expects NSImage as parameter and returns normalized point if it was found or (0,0) if it wasn't.

initWithSize:

Initialize with given resolution of camera 

-(id)initWithSize:(NSSize)aSize;
Discussion

After calling this method, image processing class is expected to be initialized with given resolution of camera
