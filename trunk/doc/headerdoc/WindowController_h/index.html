WindowController

Main clas of application.

Discussion

This is the main class of application. This class connects all parts of the applications together. Methods of this class are all actions called from GUI.


Methods

-Calibrate:

Starts the calibration

-procSettingsClicked:

Shows settings dialog of selected class.

-RunAndPause:

Starts and pauses the game.

-transformSettingsClicked:

Shows settings dialog of selected class.

-viewChooseChanged:

Changes the fullscreen view.

-viewSettingsClicked:

Shows settings dialog of selected class.
