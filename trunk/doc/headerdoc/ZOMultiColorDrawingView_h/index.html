ZOMultiColorDrawingView

View for multi color drawing

Discussion

This view is used for drawing with three colors. It is driven by ZOMultiColorDrawingController. It draws three NSBezierPath elements with different colors, squares for changing color of drawing, pausing and resetting the drawing.

This property is used to get the drawed paths. It contains three NSBezierPaths, one for each color.
