ZOBaloon

Class for storing balloons

Discussion

This class is designed to store almost everything about balloons, which are used in ZOShootingView as object for shooting.

initWithOrigin:Size:Speed:andColor:

Inits balloon with given arguments 

-(id)initWithOrigin:(NSPoint) anOrigin Size:(float) aSize Speed:(float)aSpeed andColor:(NSColor *)aColor;
Parameters
anOrigin

The point, where will the balloon appear. In screen coordinates.

aSize

Size of the balloon.

aSpeed

Speed of the ballon. In pixels per one iteration.

aColor

NSColor * with color of balloon.

Discussion

This method is used to initialize one balloon. It has all needed parameters to determine all attributes of ballon

move

Tells balloon to move 

-(void)move;
Discussion

After calling this method, balloon moves up its speed.

shooted

Tells ballon that it was shooted. 

-(void)shooted;
Discussion

This method is used to tell balloon, that the laser pointer shooted into the balloon. Balloon increments its shoot counter, which is accessible with the shots property.

Number of shots.
