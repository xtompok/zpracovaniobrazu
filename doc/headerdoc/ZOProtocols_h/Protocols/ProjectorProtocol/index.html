ProjectorProtocol

Protocol for viewing points

Declared In: ZOProtocols

Discussion

This protocol lists all methods needed to communicate with viewing classes.


Methods

-goFullscreen

Tells class to go fullscreen

-leftFullscreen

Tells class to left fullscreen

-setPoint1:

Sets new lighted point

-setPoint2:

Sets new lighted point

-showSettingsWindow

Opens settings window of the viewing class

goFullscreen

Tells class to go fullscreen 

-(void)goFullscreen;
Discussion

Because the classes for viewing uses fullscreen, this method is called when the viewing class changed and tells class that the screen is its.

leftFullscreen

Tells class to left fullscreen 

-(void)leftFullscreen;
Discussion

Because the classes for viewing uses fullscreen, this method is called when the viewing class changed and tells class to clean the screen for another class.

setPoint1:

Sets new lighted point 

-(void)setPoint1:(NSPoint)aPoint;
Parameters
aPoint

Normalised point. If the point was not found in image, aPoint is (0,0)

Discussion

This method is used to inform viewing classes that the new image was processed.

setPoint2:

Sets new lighted point 

-(void)setPoint2:(NSPoint)aPoint;
Parameters
aPoint

Normalised point. If the point was not found in image, aPoint is (0,0)

Discussion

This method is used to inform viewing classes that the new image was processed.

showSettingsWindow

Opens settings window of the viewing class 

-(void)showSettingsWindow;
Discussion

This method tells class to open the settings window. If it doesn't have the window, nothing should be done.
