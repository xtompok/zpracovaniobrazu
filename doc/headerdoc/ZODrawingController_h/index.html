ZODrawingController

Controller for drawing.

Discussion

This is a controller for one color drawing. It conforms to Projector protocol and has its GUI for configuration the drawing. It also can save drawed images to disk to process with bezpath2svg.pl.


Methods

-resetDrawing:

Resets drawing

-saveBezier:

Saves drawed path to disk.

-setWidth:

Sets width of drawed line

resetDrawing:

Resets drawing 

-(IBAction)resetDrawing:(id)sender;
Discussion

Tells ZODrawingView to reset current drawed path.

saveBezier:

Saves drawed path to disk. 

-(IBAction)saveBezier:(id)sender;
Discussion

Opens dialog for saving files and saves description of currently drawed path to provided file. This colud be used for conversion to SVG using bezpath2svg.pl.

setWidth:

Sets width of drawed line 

-(void)setWidth:(float)aWidth;
Discussion

Tells ZODrawingView to change width of drawed line to given value.
