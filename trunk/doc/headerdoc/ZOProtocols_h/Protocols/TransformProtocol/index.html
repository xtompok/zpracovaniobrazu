TransformProtocol

Protocol for transforming point coordinates

Declared In: ZOProtocols

Discussion

This protocol lists all methods needed to communicate with point transforming classes


Methods

-initWithCalibrationArray:

Initialize with given array of points

-setCalibrationArray:

Sets new calibration array

-transformPoint:

Transforms given point

initWithCalibrationArray:

Initialize with given array of points 

-(id)initWithCalibrationArray:(NSArray *)calArray;
Parameters
calArray

Array of four ZOPoints in order upper left, upper right, lower right and lower left on the dataprojector.

Discussion

After calling this method, point tranforming class is assumed as initialized with given array of points. This methods uses calibration array to compute constants for calibration equations.

setCalibrationArray:

Sets new calibration array 

-(void)setCalibrationArray:(NSArray *)calArray;
Discussion

This method sets new calibration array of the same format as init and computes new constants for calibration equations.

transformPoint:

Transforms given point 

-(NSPoint)transformPoint:(NSPoint)point;
Discussion

This methods transforms given point and transforms it in transforming equations. Returns transformed point. If the given point is (0,0), it returns (0,0)
