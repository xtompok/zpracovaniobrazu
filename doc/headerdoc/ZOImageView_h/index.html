ZOImageView

View for showing image from camera

Discussion

This View is used to view image from camera and calibration points found in that image. It is also showed the found point from laser pointer.


Methods

-crossAtPoint:

Makes a cross.

-setCalPoints:

Sets calibration array

-setImage:

Sets image for viewing

-setPoint:

Sets found point

crossAtPoint:

Makes a cross. 

-(NSBezierPath *)crossAtPoint:(NSPoint)aPoint;
Discussion

This method makes a NSBezierPath with cross on given coordinates. The path is returned.

setCalPoints:

Sets calibration array 

-(void)setCalPoints:(NSArray *)anArray;
Parameters
anArray

NSArray of four ZOPoints in this order on dataprojector: upper left, upper right, lower right, lower left.

Discussion

Sets array of calibration points, which will be showed on the image from camera as blue squares.

setImage:

Sets image for viewing 

-(void)setImage:(NSImage *)anImage;
Discussion

Sets an image from camera. It doesn't care on resolution image will be resized.

setPoint:

Sets found point 

-(void)setPoint:(NSPoint)aPoint;
Discussion

Sets a point found in image. Point is normalised and it is shown as a red cross in the image.
