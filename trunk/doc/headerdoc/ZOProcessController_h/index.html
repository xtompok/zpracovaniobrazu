ZOProcessController

Controller for ZOProcessImage

Discussion

This controller is used for communication with ZOProcessImage class. It has a GUI for setting minimal and maximal brightnesses of point and sum squares and saving and loading this data.

dictionaryWithConfigValues

Makes a dictionary from settings. 

-(NSMutableDictionary *)dictionaryWithConfigValues;
Discussion

This method takes all config variables and saves it as NSNumber objects into NSMutableDictionary object, which is returned.

handleShowSettingsWindow:

Shows settings window. 

-(void)handleShowSettingsWindow:(NSNotification *)aNotify;

loadDataFromDisk:

Loads the settings from disk 

- (IBAction) loadDataFromDisk:(id) sender;
Discussion

This method takes a dictionary of settings from given path from method pathForDataFile and sets the variables to the loaded values.

pathForDataFile

Returns path, where to save the settings 

- (NSString *) pathForDataFile;
Discussion

This method returns the NSString with path in Application Support folder, where the settings could be saved.

saveDataToDisk:

Saves the settings to disk 

- (IBAction) saveDataToDisk:(id) sender;
Discussion

This method makes a dictionary from settings using method dictionaryWithConfigValues and saves it to the path returned from method pathForDataFile.
