ZOCalibrateView

View for calibration

Discussion

This view is used to show calibration points or blank the screen.


Methods

-setCalPoint:

Sets the calibration point

setCalPoint:

Sets the calibration point 

-(void)setCalPoint:(int)index;
Parameters
index

The index of calibration point: 0 -- upper left, 1 -- upper right, 2 -- lower right, 3 -- lower left, other -- blank screen

Discussion

When this method is called, the calibration point is showed
