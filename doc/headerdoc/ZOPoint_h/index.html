ZOPoint

Simple object to store point.

Discussion

This class is used, when it is needed to use points as objects (i.e. in NSArray), so the standard NSPoint can't be used.


Methods

-description

Returns NSString with formated coordinates.

-initWithPoint:

Init with given NSPoint

-pointValue

Returns its value as NSPoint

-setPoint:

Sets internal value to given NSPoint
