ZOShootingView

View class for shooting balloons.

Discussion

This class implements the balloon shooting game. In the class are included methods for drawing and application logic.

crossAtPoint:

Makes a cross path 

-(NSBezierPath *)crossAtPoint:(NSPoint)aPoint;
Discussion

This methods returns a NSBezierPath with cross on the given coordinates.

insertBalloonAtIndex:

Inserts balloon into balloon array. 

-(void)insertBalloonAtIndex:(int)i;
Discussion

This method generates new balloon and saves it on the selected position in balloonsArray.

randFrom:to:

Random float generator 

-(float)randFrom:(float)a to:(float)b;
Discussion

Generates random float in given range.

resetGameWithData:

Resets the game with given data 

-(void)resetGameWithData:(GAMEDATA *) aData;
Discussion

Gets the data from given GAMEDATA structure, saves it in the global variables, prepare new balloon array, resets score, invalidates timer and pauses the game.

setPaused:

Pauses or unpauses the game 

-(void)setPaused:(BOOL) isPaused;
Discussion

If the want to be paused, the timer is invalidated, if not, the timer is started.

setPoint1:

Sets the first found point 

-(void)setPoint1:(NSPoint) aPoint;
Discussion

This method is called, when the point is found. The method transforms the point into screen coordinates. It checks, if the point is in balloon and increments the number of shots for shooted balloons.

setPoint2:

Sets the second found point 

-(void)setPoint2:(NSPoint)aPoint;
Discussion

This method trasforms the point into screen coordinates and saves it.

timerExpired:

Preparing things for draw 

-(void)timerExpired:(NSTimer *)aTimer;
Discussion

This method is called every time, when the timer expires. It moves all balloons, replace the lost balloons with new and replace shooted balloons and increments score.

Typedefs

GAMEDATA

Structure for game data

GAMEDATA

Structure for game data 

typedef struct { 
    int maxBalloons; 
    float maxSpeed; 
    int maxLost; 
    float delay; 
    int maxShots; 
    int minSize; 
    int maxiSize; 
} GAMEDATA;
Fields
maxBalloons

The maximum number of balloons on the screen.

maxSpeed

The maximum speed of balloon. In pixels per cycle.

maxLost

The maximum number of lost balloons.

delay

The delay between redarwing the screen and moving balloons.

maxShots

The number of shots needed to destroy the balloon.

minSize

The minimum size of balloon. In pixels.

maxiSize

The maximum size of balloon. In pixels.

Discussion

This structure is used for send the game data from controller and could be used for saving game data. In this strucutre there are all settings usable in game.
